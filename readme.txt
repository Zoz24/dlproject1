For the training part of the code, run it on google collab.
1. First mount the data from Google Drive,
2. then create a code block for all the imports
3. Create a block for all parameters
4. Create a block for load_dataset function
5. Create a block to prepare and train data
5. Create a block for the CNN structure
6. Compile the model and use callback to save the model as h5
7. Run the model

For the testing part of the code, also run it on google collab.
1. Mount the data from Google Drive
2. Create a code bloakc for process_path and transformation function
3. Process the test images in a new block and load the saved model
4. Run the prediction
